Interactive Geometry

Edexcel IAL Further Pure Mathematics F3

A standard way to introduce conics is to start from simple ingredients such as a focus and a directrix, then use a defining rule to create familiar curves like ellipses and hyperbolas. On this page, we start with curves that are already known and use moving tangent, perpendicular, and intersection rules to generate new loci from them.

A useful analogy is this: the original curve is the raw material, while the geometric rule is the production method. As a point moves along the given curve, the same rule is applied again and again, and the output point traces a new graph or curve.

Example 1

Curve: \[ x = ct, \qquad y = \frac{c}{t}. \]
Tangent at \[ Q\!\left(ct,\frac{c}{t}\right) \] is \[ x + t^2 y = 2ct. \]
If \(P\) is the foot of the perpendicular from the origin \(O=(0,0)\) to this tangent, then the locus of \(P\) is \[ (x^2+y^2)^2 = 4c^2xy. \]
The display traces the moving point \(P\) while the original point \(Q\) moves along the given parametric curve.

\(t\): 1.00

Example 3

Ellipse: \[ E:\quad \frac{x^2}{a^2}+\frac{y^2}{b^2}=1. \]
At \[ P(a\cos\theta,\; b\sin\theta), \] the tangent \(l_1\) is \[ \frac{x\cos\theta}{a}+\frac{y\sin\theta}{b}=1. \]
Circle: \[ C:\quad x^2+y^2=a^2. \] At \[ Q(a\cos\theta,\; a\sin\theta), \] the tangent \(l_2\) is \[ x\cos\theta + y\sin\theta = a. \]
Solving the two tangent equations leads to four logical cases: \[ \cos\theta\neq 0,\ \sin\theta\neq 0 \implies R=l_1\cap l_2=(a\sec\theta,0), \] \[ \cos\theta\neq 0,\ \sin\theta=0 \implies l_1=l_2,\quad x=\pm a, \] so there are infinitely many intersection points on a vertical line; \[ \cos\theta=0,\ \sin\theta\neq 0 \implies l_1: y=\pm b,\quad l_2: y=\pm a, \] hence there is no intersection if \(a\neq b\), and infinitely many if \(a=b\). The case \(\cos\theta=0\) and \(\sin\theta=0\) is impossible.
Use the two checkboxes to choose which case to display. In the infinite-intersection cases, the full blue locus line is drawn directly.
